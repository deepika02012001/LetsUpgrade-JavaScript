Local variables are those variables which can be accessed from inside the function where they are declared. Variables declared with the 'let' keyword can have block scope{}.Variables declared inside a block{} cannot be accessed from outside the block.

Example:
{
let x=10;
document.getElementById('demo3').innerHTML=x;
}
document.getElementById('demo4').innerHTML=x;

Output:


Redaclaring a variable using let can cause problems and redeclaring a variable inside a block will not redeclare the variable outside the block.

Example:
var z=20;
{
let z=10;
document.getElementById('demo7').innerHTML=z;
}
document.getElementById('demo8').innerHTML=z;
