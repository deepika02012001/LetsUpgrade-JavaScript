Different console functions:

1.console.assert()-This writes an error message to the console if the assertion is false.
2.console.clear()-It clears the console.
3.console.count()-It logs the number of times that this particular call to count() has been called.
4.console.error()-This outputs an error message to the console.
5.console.group()-It creates a new inline group in the console.This indents following console messages by an additional level, until console.groupEnd() is called.
6.console.groupCollapsed()-It creates a new inline group in the console.This new group is created collapsed.A disclosure button has to be used to expand it.
7.console.groupEnd()-It exits the current inline group in the console.
8.console.info()-It outputs an informational message to the console.
9.console.log()-It outputs a message to the console.
10.console.table()-Displays tabular data as a table.
11.console.time()-It starts a timer to track how long an operation takes.
12.console.timeEnd()-It stops the timer that was started by console.time().
13.console.trace()-It outputs a stack trace to the console.
14.console.warn()-This outputs a warning message to the console.
