Datatypes in JavaScript:

Javascript variables can hold many data types:numbers,strings,objects and more:

1)Number

var a=10;

2)strings

var name="john";

3)boolean

var abc=false;

4)object

var a={name:"reena",age:34};

5)Undefined

var x;
//Not initializing any value to x

6)Null

var x=null;

7)strings in quotes

var str="Hi";
var str1="Hi all";
var str2="Hi 'John'";
var str1='Hi all';

8)decimals

var n=32.00;

9)exponential numbers

var z=123e5;
var z=123e-5;

10)arrays

var fruits=['apple','mango','banana'];
