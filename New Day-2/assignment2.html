String Methods

String="Good things come to those who wait"

search()-searches the string of specified value and returns the position of match.

var pos=str.search("who");

var pos1=str.search("where");

slice()-It extracts a part of a string and returns the extracted part as a new string.It takes two parameters:start position and end position.

var res=str.slice(3,12);

Negative values are given to start slicing from end of string:
var res1=str.slice(-6,-3);

If second parameter ommitted, it will print the rest of the string:
var res2=str.slice(5);

Similarly negative single parameter to start from end:
var res3=str.slice(-7);

substring()-It extracts a part of a string and returns the extracted part as a new string.It does not accept negative indexing.

var tmp=str.substring(4,11);

substr()-It extracts a part of a string and returns the extracted part as a new string.It takes two parameters.Second parameter specifies the length of extracted part.

var n=str.substr(3,5);

If second parameter ommitted, it will print the rest of the string:
var n1=str.substr(3);

If first parameter is negative , position counts from end of string:
var n2=str.substr(-3);

replace()-replaces a specified value by another value in string.It replaces only the first match.It is case sensitive.

var p1=str.replace("wait","work hard");

concat()-joins two or more strings

var newstr="hi";
var newstr1="hello";
var newstr2=newstr.concat(" ",newstr1);

trim()-removes whitespace from both sides of string

var s=" hi ";
var s1=s.trim();

charCodeAt()-returns unicode of character at specified index in a string

var st="hello all";
var st1=st.charCodeAt(4);

Access property at an index

var sp="Good";
var sp1=sp[2];

Array Methods

var flowers=['rose','lotus','sunflower','hibiscus','lily'];

toString()-converts array to string of array values seperated by comma.

flowers.toString();

join()-joins all the array elements by a specified seperator.

flowers.join(" * ");

changing array elements using index

flowers[1]="tulip";

length property-to append new element to array

flowers[flowers.length]="jasmine";

delete-delete array elements

delete flowers[2];

splice()-add new items to array.First parameter defines position where new element should be added.second parameter tells how many elements should be removed.Rest of them define new elements to be added.

flowers.splice(1,2,"marigold","daisy");

splice()-to remove element.First parameter defines position where new element should be added.second parameter tells how many elements should be removed.

flowers.splice(0,1);

sort()-sort based on alphabets

flowers.sort();

reverse()-reverses the array

flowers.reverse();
